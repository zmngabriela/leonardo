# Leonardo Cadore Website

Portfolio website developed for a content creator, photographer, and filmmaker, with a strong visual focus on media presentation, motion, and storytelling.

## Overview

The project was designed to highlight visual work through a more immersive experience, combining large media assets, animations, and a clean structure to support the creator’s identity and portfolio.

## Tech Stack

- React
- JavaScript
- CSS

## Features

- Responsive portfolio layout
- Large media sections for video and visual content
- Animated transitions and interactions
- Visual storytelling focused on creative work
- Structured presentation of portfolio content

## Purpose

This project was built to create a portfolio experience that feels visually engaging while still keeping navigation clear and intuitive.
